## Running Locally

1. Install dependencies using pnpm

```sh
pnpm install
```

2. Copy `.env.local.sample` to `.env.local` and update the variables

```sh
cp .env.local.sample .env.local
```

3. Start the development server

```sh
pnpm dev
```
